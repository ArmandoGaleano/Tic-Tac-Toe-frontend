Para jogar o jogo da velha [Clique aqui](https://tictactoefrontend.herokuapp.com/)

## Instalar na Máquina

No diretório do jogo

### `npm install`

Para instalar os pacotes.

### `npm run start`

Abra http://localhost:3000 para ver o jogo no browser
